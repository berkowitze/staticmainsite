This is a list of code projects I have worked on.

  • github.com/hveto/hveto - HVeto, or 'Hierarchical Veto', is a method used at LIGO to rate how significantly correlated noise in channels devoted to external sourses (for instance quantum or seismic) is to noise in the main gravitational wave detector. Signals can be filtered out using this measure of correlation. I helped to implement and improve this method.
